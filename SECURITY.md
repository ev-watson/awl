# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 0.3.x | Yes |
| < 0.3.0 | No |

## Reporting a Vulnerability

Please do not open public issues for security-sensitive bugs.

Report vulnerabilities by contacting the maintainer at `etwatson@ucsd.edu` with:

- A clear description of the issue
- Reproduction steps or a proof of concept
- The affected version or commit
- Any proposed mitigation, if you have one

You should receive an acknowledgment within a few business days. Fixes will be prepared privately and disclosed once a patch is available.
